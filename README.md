# react-lib-nowports

> Library global for Nowports Inc.

## Node

```bash
use node>=12
```

## Install

```bash
npm install
```

## Run

```bash
npm start
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'react-lib-nowports'

class Example extends Component {
  render() {
    return <MyComponent {...props}/>
  }
}
```
